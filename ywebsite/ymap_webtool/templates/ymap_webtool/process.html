STEP 1: Mutation at DNA level to protein level (optional)

When the input file contain mutations at DNA level, it is translated to protein level for yMap to be able to process. All the non-synonymous mutation are reported in a file that will be use as input by the core function of yMap.

STEP 2: Mapping the mutations to protein features and post translational modifications.

The mutations at protein level are mapped to protein features and post translational modifications. yMap generate a report file for each types of protein features and post translational modifications that contains/overlaps with mutations from the input file.

Protein features:

  • proteins domains
  • proteins-nucleotide binding domains
  • proteins structural regions
  • proteins active and binding sites

Post translational modifications:

  • PTM present at the interface of two proteins and known to play role in interaction (Beltrao et al. Cell 2012)
  • PTM present at the interface of two proteins and known to play role in interaction (Beltrao et al. Cell 2012)
  • PTMs (predicted) involved in the crosstalk within a given protein at baker's years (Minguez el 2012)
  • PTMs (predicted) involved in the crosstalk in different proteins at baker's years (PTMcode 2.0; Minguez el 2012)
  • PTMs containing motifs in a close proximity are named hotspots (Beltrao et al. Cell 2012)

STEP 3: GO enrichment analysis

A Go enrichement analysis is performed for thes set of proteins containing mutation located in functional region. It uses the function get_enriched_terms from the orange bioinformatics package. The p-value of the Go enrichement analysis is calculated against the background of the complete genome (Demšar et al., 2013).

STEP 4: Biogrid id

yMap report the biogrid id for each mutated protein. The Biological General Repository for Interaction Datasets (BioGRID) is a public database that archives and disseminates genetic and protein interaction data from model organisms and humans.

STEP 5: Final report

All the analyses are summarized in a ‘final-report’ with the proteins that have variants in different above mentioned protein functional regions, the type of mutation (synonymous or non- synonymous), feature, feature type and source of data. The report is complemented with enrichments file and a BioGrid ids containing file.
